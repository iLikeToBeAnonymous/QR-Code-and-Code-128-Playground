Important Notes:

Because the QRCode script relies heavily on a variety of classes, it is important that the script be declared before any code that uses it.
Therefore, if the QRCode script is in-lined in the HTML doc, it should be done before the rest of the document. If instead it is placed in a CDN, it should be declared in the head of the document, just like the declaration of the src for jQuery (as shown in this document).
